# BDD-Quotes-Automation
This project is a Behavior-Driven Development (BDD) based test automation framework built for the Quotes website. It uses Python, Selenium WebDriver, and Behave to automate UI testing scenarios in a structured and readable format.

The framework follows the BDD approach where test cases are written in simple Gherkin syntax (Given-When-Then), making it easy to understand for both technical and non-technical users.
